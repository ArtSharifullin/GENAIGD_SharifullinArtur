
## Design Constants (HW06) — Moonlit Harvest: Хранитель Грани

### 1) Константы персонажа (Hero / Элиан — Хранитель Грани)

- **Силуэт**: прямая, но расслабленная осанка (фермерская, не воинская). В ночном режиме — лёгкая сутулость от усталости
- **Голова/лицо**: соломенная шляпа (днём) / капюшон с лёгким свечением глаз (ночью). Лицо частично скрыто, без ярко выраженной мимической пластики (pixel art минимализм)
- **Ключевая деталь**: лунный талисман в зоне груди — пульсирует `#ffa800` в ночное время
- **Костюмный мотив (ночь)**: длинный тёмно-синий плащ (`#1a2a4a`) с лунной вышивкой, простые ремни, кожаные перчатки (опционально)
- **Костюмный мотив (день)**: простая рабочая рубашка и штаны (`#8b5a2b` и `#4a3728`), соломенная шляпа, лунная мотыга за спиной
- **Цветовой акцент**: ограниченный янтарный свет `#ffa800` от талисмана и глаз (ночью) / тёплые земляные тона (днём)

### 2) Константы мира / стиля

- **Палитра**:
  - День: тёплые земляные тона (`#8b5a2b`, `#4a3728`, `#6b8e23`), мягкие тени
  - Ночь: тёмный фон (`#191c21`), глубокий синий (`#1a2a4a`), локальные акценты `#ffa800` (лунный свет)
  - Акценты: `#88ccff` (небесный — для магии и кристаллов), `#44ff44` (здоровье), `#ff4444` (опасность)

- **Материалы**:
  - День: грубая ткань, дерево, солома, матовые поверхности
  - Ночь: матовая сталь (клинок), плотная ткань (плащ), полупрозрачные кристаллы
  - Минимум глянца — только полированное золото для артефактов

- **Свет**:
  - День: мягкий рассеянный свет, тёплые тона
  - Ночь: локальный key/rim от талисмана и лунных цветов, без сильного bloom
  - Идеальное уклонение: золотая вспышка `#ffa800` с эффектом замедления

- **Камера**:
  - Иконки предметов: фронтальная симметрия
  - Персонажи: 3/4 или полупрофиль
  - Локации: top-down изометрия (16x16 тайлы)

- **Стилизация**:
  - Чёткий pixel edge (Nearest Neighbor, без сглаживания)
  - Стабильная плотность пикселей (32x32 для базовых спрайтов)
  - Dithering для градиентов (вместо blur)

### 3) Что ломает consistency

| Нарушение | Почему |
| :--- | :--- |
| ❌ Сильный soft/painterly рендер вместо pixel структуры | Нарушает Pixel Art стиль, утверждённый в Style Bible |
| ❌ Случайные яркие цвета и кислотные акценты (кроме `#ffa800`) | Разрушает тёмную палитру `#191c21` |
| ❌ Глянцевый/пластиковый материал брони/предметов | Материалы должны быть матовыми или с мягким свечением (кристаллы) |
| ❌ Потеря расслабленной/сутулой осанки | Разрушает образ «уставшего фермера-хранителя» |
| ❌ Исчезновение лунного талисмана на груди | Ключевой визуальный маркер, связывающий день и ночь |
| ❌ Размытый outline из-за избыточного fog/bloom | Нарушает чёткость пиксельных границ |
| ❌ Самопроизвольная смена реквизита (мотыга/меч/плащ) без запроса | Нарушает вариативность, заданную в Hero Card |
| ❌ Отсутствие различий между дневным и ночным режимами | Два режима игры должны визуально различаться |
| ❌ Воинская прямая осанка | Персонаж — фермер, а не рыцарь |

### 4) Допустимая вариативность (Allowed variability)

| Категория | Что можно менять |
| :--- | :--- |
| **Эмоция** | Нейтральный / Напряжённый / Уставший (в пределах pixel art) |
| **Ракурс** | 3/4 (основной) / Профиль / Полуоборот |
| **Свет** | Интенсивность свечения талисмана (от слабого до яркого `#ffa800`) |
| **Экипировка** | Степень износа (чистый / в пыли / после боя). Лёгкие вариации: перчатка, подсумок |
| **Состояние** | Здоров / Ранен (тёмные пятна, тусклое свечение) |
| **Время суток** | Дневной вариант (соломенная шляпа, рабочий костюм) / Ночной вариант (плащ, капюшон) |
