
## Moonlit Harvest: Хранитель Грани

**Цель:** список ассетов под выбранный baseline (гибрид Direction A + Direction B из `style/style-bible.md`).

**Baseline-константы:**
- Палитра: #191c21 (фон), #ffa800 (лунный акцент), #88ccff (небесный), #44ff44 (здоровье), #ff4444 (опасность)
- Материалы: полупрозрачные кристаллы + полированное золото (оправа)
- Свет: мягкое внутреннее свечение (#ffa800), рассеянный днём, контрастный ночью
- Камера: фронтальная симметрия (иконки), изометрия (локации), 3/4 (персонажи)
- Степень стилизации: Pixel Art 32x32 (базовый), 64x64 (боссы), 96x96 (Лорды)

### Следующий шаг (после HW05)

1. **Провести mini-серию по A-baseline** — сгенерировать 3-4 варианта Игрока (A01) в профиль и полуоборот для утверждения ракурсов
2. **Сгенерировать 2 локационных кадра** (A03 и A05) с теми же стилевыми правилами — проверить масштабируемость палитры на окружение
3. **Подготовить короткий ref-sheet героя** (front / 3-4 / side / dodge animation) на базе selected outputs
4. **Обновить промпт-скелет** под задачу “character + location consistency” — добавить блоки `mood`, `camera`, `palette` в фиксированном порядке
5. **Сгенерировать A07 (Лунная пыльца)** — ресурс будет использоваться в UI постоянно, нужна высокая читаемость в 24x24 и 32x32
6. **Зафиксировать запреты** для врагов (A08): никаких реалистичных деталей, только силуэт и глаза
