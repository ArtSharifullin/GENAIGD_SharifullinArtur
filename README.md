# Moonlit Harvest: Хранитель Грани

Жанр: Farming Simulator / Life Simulator / Action-RPG / Light Management.  
USP: днем игрок развивает ферму и связи с жителями, ночью уходит в лунный лес с тайминг-боевкой и риском потери ресурсов.  
Целевая аудитория: фанаты уютных фермерских игр, игроки, которым нужен напряженный ночной режим, любители пиксель-арта.  
Платформы: PC, Mobile (iOS/Android), Web (WebGL).

Куда смотреть в первую очередь:
- `GDD.md` - основной дизайн-документ проекта.
- `logs/gdd-log.md` - журнал итераций и решений по геймдизайну.
- `logs/workflow-notes.md` - заметки по генерациям, параметрам и отбору.
- `style/Style-bible.md` - визуальные правила и baseline стиля.
- `outputs/selected/` — отобранные результаты генераций (W04–W08).
- `images/selected/` — ранние SDXL-кадры (W04, seed 42424242).
- `shots/shot-list.md` — скрингены и Control plan.
- `logs/comfyui-missing-assets-guide.md` — промпты для недостающих генераций в ComfyUI.
