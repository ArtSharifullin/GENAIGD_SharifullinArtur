## Character refs (HW06) — Moonlit Harvest: Хранитель Грани

### Роль персонажа
- **Кто это**: Хранитель Грани **Элиан (Elian)** — базовый герой проекта
- **Зачем нужен в игре**: визуальный baseline персонажа для генераций и для «лица» проекта; связующее звено между дневной фермой и ночным лесом
- **Функция в геймплее/нарративе**: днём — фермер, заботящийся об урожае и общающийся с жителями; ночью — хранитель, сражающийся с тенями и собирающий Лунную пыльцу для восстановления Сердца Луны

### Визуальные ориентиры (в рамках baseline из Style Bible)

- **Силуэт**: расслабленная фермерская осанка (не воинская). В ночном режиме — лёгкая сутулость от усталости. Длинный плащ с капюшоном (ночью) / рабочая одежда без плаща (днём)
- **Голова/лицо**: соломенная шляпа (днём) / капюшон с лёгким свечением глаз (ночью). Лицо частично скрыто, без ярко выраженной мимики
- **Ключевая деталь**: лунный талисман на груди — пульсирует `#ffa800` в ночное время
- **Костюм (ночь)**: тёмно-синий плащ (`#1a2a4a`) с простой лунной вышивкой, кожаные ремни, лунная мотыга или клинок за спиной
- **Костюм (день)**: простая рабочая рубашка (`#8b5a2b`), штаны (`#4a3728`), соломенная шляпа, лунная мотыга как инструмент
- **Материалы**: матовая ткань/кожа (плащ), грубое дерево/сталь (инструменты), янтарное свечение (талисман)
- **Цветовые акценты**: доминанта `#191c21` (ночь) или тёплые земляные тона (день), акцент `#ffa800` (талисман, глаза, лунный свет), `#88ccff` (небесный — для магии)
- **Настроение**: днём — спокойное, уютное; ночью — напряжённое, мрачное, но с намёком на надежду (золотой свет)
- **Степень стилизации**: pixel-art (32x32 базовый спрайт), чёткие формы, читаемый силуэт, минимум «шума»

### Выводы по визуальным константам (3–4)

1. **Два режима, два узнаваемых силуэта**:
   - День: соломенная шляпа, рабочая одежда, лунная мотыга
   - Ночь: капюшон, плащ, лунный талисман, светящиеся глаза

2. **Ключевой маркер идентичности** — лунный талисман на груди с пульсацией `#ffa800`. Он должен быть виден и днём, и ночью

3. **Акцент один** — `#ffa800` (лунный свет). Вторичный акцент `#88ccff` — только для магии и кристаллов. Никаких конкурирующих ярких деталей

4. **Pixel Art стиль** — стабильная плотность пикселей (32x32 для базового спрайта), чёткий outline, минимум шума. Днём — чуть мягче, ночью — жёстче и контрастнее

### Запреты (Bans)

| Запрет | Почему |
| :--- | :--- |
| ❌ Яркая/кислотная палитра (кроме акцента `#ffa800` и `#88ccff`) | Нарушает тёмную палитру `#191c21` |
| ❌ Аниме-черты лица, огромные «милые» глаза | Персонаж — взрослый хранитель, не аниме-герой |
| ❌ Реалистичная кожа/фоторендер/3D-look | Нарушает Pixel Art стиль |
| ❌ Воинская прямая осанка | Элиан — фермер, а не рыцарь |
| ❌ Случайные узоры на плаще, вышивка, дешёвый «фэнтези-декор» | Нарушает лаконичность и читаемость силуэта |
| ❌ Слишком крупные наплечники/рога/крылья | Артефакты из другой стилистики |
| ❌ Много мелкой бижутерии и декора | Ломает читаемость на малом размере (32x32) |
| ❌ Исчезновение лунного талисмана на груди | Ключевая деталь идентичности |
| ❌ Размытые контуры / избыточный fog | Нарушает чёткость пиксельных границ |
| ❌ Отсутствие визуального различия между днём и ночью | Два режима игры должны различаться |
