
## Location refs (HW06) — Moonlit Harvest: Хранитель Грани

### Что это за локация
- **Название (рабочее)**: Лунный лес — Светящаяся поляна
- **Где находится**: основная ночная локация, доступная после активации Лунного пути (20:00 – 06:00)
- **Зачем нужна**: место для сбора Лунной пыльцы, ночных культур, боёв с тенями и встреч с духами леса
- **Какое чувство должна вызывать**: таинственное напряжение, смешанное с благоговением перед лунным светом; одновременно красиво и опасно

### Ключевые рефы (3)

| # | Референс | Что берём | Чего избегаем |
| :--- | :--- | :--- | :--- |
| **Ref 01** | Hyper Light Drifter (лесные локации) | Тёмные силуэты деревьев, локальные акценты цвета, ограниченная палитра, свечение | Кислотные цвета, избыточный bloom |
| **Ref 02** | Tunic (ночной лес) | Лунный свет как источник, читаемая геометрия, мягкое свечение кристаллов | Перегруженность деталями, мультяшность |
| **Ref 03** | Eastward (подземные/лесные уровни) | Контраст между тьмой и источниками света, уютное освещение в опасном месте | Избыточный декор, слишком яркую палитру |

### Связь с персонажем

- **Палитра локации** — доминанта `#191c21` (фон) + акценты `#ffa800` (лунные цветы, алтари) и `#88ccff` (кристаллы). Не перекрикивает персонажа, а дополняет его
- **Талисман героя** (`#ffa800`) — ключевой визуальный мост между персонажем и миром. Свет талисмана должен перекликаться со светом лунных цветов на поляне
- **Читаемость силуэта** — силуэт героя должен оставаться чётким на фоне тёмного леса. Туман войны не должен «съедать» контуры
- **Контраст стена/фон** — достаточно высокий для сохранения читаемости персонажа в режиме боя

### Дополнительные визуальные ориентиры

| Аспект | Требование |
| :--- | :--- |
| **Цветовая палитра** | `#191c21` (фон, деревья), `#1a2a4a` (глубокий лес), `#ffa800` (лунные цветы, алтари, Источники света), `#88ccff` (лунные кристаллы, магия) |
| **Освещение** | Источник света — Луна (сверху) + лунные цветы и кристаллы (локально). Мягкое свечение, без резких теней |
| **Камера** | Top-down изометрия (как в Stardew Valley), вид сверху под углом. Персонаж в центре, обзор 5x5 клеток (плюс туман войны по краям) |
| **Детализация** | Высокая для ключевых объектов (алтари, кристаллы), низкая для фона (силуэты деревьев). Pixel Art 16x16 тайлы для окружения |
| **Туман войны** | По краям экрана — плотная тьма `#191c21` с постепенным переходом к освещённой зоне |

### Запреты для локации

| Запрет | Почему |
| :--- | :--- |
| ❌ Яркое дневное освещение | Локация — ночная, должна быть тёмной |
| ❌ Кислотные/насыщенные цвета (кроме `#ffa800` и `#88ccff`) | Нарушает палитру `#191c21` |
| ❌ Избыточный декор (статуи, фонтаны, сложные узоры) | Локация — дикий лес, а не дворец |
| ❌ Реалистичные текстуры деревьев/земли | Нарушает Pixel Art стиль |
| ❌ Сглаживание (blur, anti-aliasing) | Нарушает чёткость пиксельных границ |
| ❌ Слишком плотный туман (не видно силуэта героя) | Ломает читаемость персонажа |
| ❌ Отсутствие лунных акцентов (`#ffa800`) | Без них локация теряет связь с тематикой игры |

### Референсы для тумана войны

- Dead Cells (эффект тьмы по краям экрана) — плавное затухание к краям, персонаж всегда в центре
- Darkest Dungeon (периферийная тьма) — напряжение и ощущение неизвестности за пределами видимого
